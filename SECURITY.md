# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Vulnerabilities Reporting

**Please DO NOT file public issues for security vulnerabilities.**

Report privately via:
- GitHub Security Advisory: https://github.com/mercuryPark/oc-setup/security/advisories/new
- Email: (contact mercuryPark for email)

We aim to provide initial response within 48 hours, and a fix plan within 7 days.
